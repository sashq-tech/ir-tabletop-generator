# IR Tabletop Generator Home

## Project Snapshot

Incident Response Tabletop Exercise Generator is a separate site idea inspired by the ReadyScene workflow, but aimed at ISSOs, security managers, compliance owners, and small IT teams.

The core product idea is simple: open the site, choose a few exercise settings, and generate a practical tabletop exercise packet without a login, backend, database, or uploaded organizational data.

## Current Direction

- Build a prototype before buying a domain.
- Keep the site static and browser-based.
- Use a serious cyber operations theme.
- Focus on meeting-ready tabletop packets, not enterprise simulation.
- Use downloaded source documents as reference material, not copied content.
- Use this Obsidian vault as the project canvas for notes, decisions, and future planning.

## Current Prototype

Prototype files live one folder above this vault:

- `../index.html`
- `../styles.css`
- `../app.js`
- `../README.md`

The current prototype generates:

- Scenario brief
- Exercise objectives
- Timed injects
- Facilitator notes
- After-action prompts
- Evidence checklist
- Copy and print output
