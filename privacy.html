Privacy notice

Privacy

Response Rehearsal is designed to run as a static, browser-only tool. The current app code does not create accounts, accept uploads, send generated exercise content to a backend, set app cookies, use analytics scripts, or store settings in browser storage.

Information you enter

Exercise settings, generated text, copied output, printed pages, and downloaded files are handled in your browser. Do not enter real credentials, confidential incident details, personal information, regulated data, sensitive infrastructure information, or anything that would create risk if shared.

Hosting logs

The planned public domain is responserehearsal.com, but it is not connected to this static site yet. A future static hosting provider may receive ordinary web request information such as IP address, browser user agent, requested page, and timestamp. That hosting-layer behavior is separate from the generator code itself and should be reviewed when a domain or hosting provider is connected.

Third-party services

The current public runtime does not include analytics, advertising, affiliate tracking, external scripts, uploads, accounts, or backend API calls. If those are added later, this page should be reviewed and updated before launch.
